## Deployment

To install the requirements

_Python:_
```bash
python manage.py install
```
_Npm:_
```bash
npm install
```

*To start:*
```bash
python manage.py start
```

Requirements:
```cmd
C:\>node-v
v14.17.x

C:\>npm -v
6.14.x

C:\>python -V
Python 3.9.x
```
